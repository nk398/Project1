The Internet: IP address and DNS


How are the devices connected over a network? Laptops or phones are connected to Wi-Fi. Wi-Fi in-turn connects to internet. These internets have service providers, also called ISP where billions of devices are interconnected. All devices on the internet have unique address which is a number. This is called the IP address, which is 32 bits long. These 32 long bits are sent as 0s and 1s in the network.

Earlier, initially the number identity of the country, regional network of device, then the sub network and finally the address of the specific device. In this manner, the branching occurred. This type is called as the IPV4 network. It came in 1973 but was adapted in 1980. Currently, we use IPV6 address which uses 128 bits per address in the network. DNS or the Domain Name System associate names like www.example.com with the corresponding addresses. The computer uses the DNS to look up the domain name and get the corresponding IP address. DNS servers are connected in a distributed hierarchy and are in-turn divided into the respective zones of domain names.
